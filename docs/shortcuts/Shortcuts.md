# IntelliJ Shortcuts

## ️ Navigation

| Aktion                   | Shortcut         | Beschreibung                                                         |
| ------------------------ | ---------------- | -------------------------------------------------------------------- |
| Gehe zu Klasse           | ⌘ O              | Öffnet Klassensuche                                                  |
| Zur Deklaration springen | ⌘ B oder ⌘ Klick | Springt zur Definition                                               |
| Suche überall            | ⇧ ⇧              | oder auch ⌘ ⇧ n Globale Suche (Klassen, Dateien, Aktionen, Settings) |
| Letzte Dateien           | ⌘ E              | Zeigt zuletzt geöffnete Dateien                                      |
| Letzte Bearbeitungen     | ⇧ ⌘ E            | Zeigt letzte Editierorte                                             |

---

## Code Editing

| Aktion                               | Shortcut | Beschreibung              |
| ------------------------------------ | -------- | ------------------------- |
| Code formatieren                     | ⇧ ⌘ L    | Formatiert Code           |
| Imports optimieren                   | ⌥ ⌘ O    | Räumt Imports auf         |
| Zeile duplizieren                    | ⌘ D      | Dupliziert aktuelle Zeile |
| Zeile löschen                        | ⌘ ⌫      | Löscht aktuelle Zeile     |
| Kommentar (mehrzeilig)               | ⌘ ⇧ 7    | Block-Kommentar           |
| Methode / Getter / Setter generieren | ⌘ N      | Öffnet Generate-Menü      |
| Parameterinfo                        | ⌘ P      | Zeigt Methodenparameter   |

---

## Refactoring

| Aktion          | Shortcut | Beschreibung                   |
| --------------- | -------- | ------------------------------ |
| Umbenennen      | ⇧ F6     | Benennt Symbol um              |
| Signatur ändern | ⌘ F6     | Ändert Parameter / Rückgabetyp |

---

## Suchen & Ersetzen

| Aktion                 | Shortcut | Beschreibung               |
| ---------------------- | -------- | -------------------------- |
| Ersetzen in Datei      | ⌘ R      | Ersetzen im aktuellen File |
| Projektweite Suche     | ⇧ ⌘ F    | Suche im Projekt           |
| Projektweites Ersetzen | ⇧ ⌘ R    | Globales Ersetzen          |
